Тіні

Візуальна глибина, реалістичність, динамічність та ефектність веб-елементів.
Створюйте 3D-ефекти, ефекти натурального світла і тіні.

box-shadow

Властивість box-shadow додає тіньові ефекти навколо елемента. Тінь описується зміщенням по X та Y відносно елемента, радіусами розмиття та розтікання, а також кольором.

Якщо для елемента з тіньовою рамкою вказано border-radius, тіньова рамка набуває таких самих заокруглених кутів.

Властивість box-shadow може мати кілька значень:

  • inset: змінює тінь на тінь всередині рамки - всередині рамки, вище фону, але нижче вмісту (наприклад, box-shadow: 5px 10px inset;)
  • <offset-x>: (обов'язковий) задає відстань по горизонталі; від'ємні значення розміщують тінь зліва від елемента; додатні - справа від елемента
  • <offset-y>: (обов'язковий) задає відстань по вертикалі; від'ємні значення розміщують тінь над елементом, додатні - під елементом
  • <blur-radius>: чим більше це значення, тим більше розмиття; від'ємні значення не допускаються (наприклад, box-shadow: 10px 10px 8px #888;). Якщо не вказано, буде встановлено значення 0, і край тіні буде різким
  • <spread-radius>: додатні значення призведуть до того, що тінь розшириться і стане більшою, від'ємні значення призведуть до того, що тінь зменшиться (наприклад, box-shadow: 10px 10px 8px 10px #888;). Якщо не вказано, то буде 0 і тінь буде такого ж розміру, як і елемент
  • <color>: задає колір тіні
  • none: значення за замовчуванням; тінь не відображається

Ви можете вказати один box-shadow, використовуючи 2-4 значення <length>, <color> та inset:

  • Якщо вказано лише 2 значення, вони інтерпретуються як значення <offset-x><offset-y>.
  • Якщо вказано 3-тє значення, воно інтерпретується як <blur-radius>.
  • Якщо задано 4-е значення, воно інтерпретується як <spread-radius>.

Щоб вказати декілька тіней, надайте список тіней, розділених комами (кожна з яких має більше значення <length>, ніж попередня), наприклад, так:

box-shadow: 5px 5px #888, 10px 10px #7eb4e2, 15px 15px #f69ec4;

Одностороння тінь

Ви можете створити односторонню тінь, встановивши значення <offset-x> на 0 і застосувавши від'ємний <spread-radius>.

box-shadow: 0 10px 5px -5px #888;

Одностороння тінь на всю ширину

Щоб створити односторонню тінь, довжина якої дорівнює довжині елемента, встановіть значення <offset-x> рівним 0 і застосуйте від'ємне значення <offset-y>:

box-shadow: 0 -16px 12px 12px #000;

Він буде працювати тільки для навігаційної панелі, яка прилипає до верхньої, лівої та правої частини сторінки. Якщо застосувати цей тип тіні до елемента, навколо якого є простір, він виглядатиме як тристороння тінь.

Тінь з двох протилежних сторін

Щоб створити тінь, яка падає з двох протилежних сторін, додайте дві тіні через кому. Друга тінь повинна містити від'ємне значення для <offset-x>.

box-shadow: 10px 0 10px -10px #666,
            -10px 0 10px -10px #666;

Нерегулярні тіні-краплі

Ось (трохи змінені) приклади створення нерегулярних тіней з книги Леа Вероу "Секрети CSS". У наведених нижче прикладах тіні створюються за допомогою властивості filter, а не box-shadow.

div {
    position: relative;
    display: inline-flex;
    flex-direction: column;
    justify-content: center;
    vertical-align: bottom;
    width: 5.9em;
    height: 5.2em;
    margin: .6em;
    background: #7eb4e2;
    filter: drop-shadow(.1em .1em .1em rgba(0,0,0,.5));
    font: 200%/1.6 "Amatic SC", cursive;
    font-weight: 700;
    text-align: center;
}

.speech {
    border-radius: .3em;
    
    &::before {
        content: "";
        position: absolute;
        top: 1em;
        right: -.7em;
        width: 0;
        height: 0;
        border: 1em solid transparent;
        border-left-color: #7eb4e2;
        border-right-width: 0;
    }
}

.dotted {
    background: transparent;
    border: .3em dotted #7eb4e2;
}

.cutout {
    border: .5em solid #7eb4e2;
    border-image: 1 url('data:image/svg+xml, <svg xmlns="http://www.w3.org/2000/svg" width="3" height="3" fill="%237eb4e2"><polygon points="0,1 1,0 2,0 3,1 3,2 2,3 1,3 0,2"/></svg>');
    background-clip: padding-box;
}

text-shadow

Властивість text-shadow корисна для додавання тіні до тексту. Тіні можуть бути одношаровими або багатошаровими, розмитими, кольоровими або прозорими. Застосовуючи тінь до елемента, ви можете вказати тільки одне значення довжини і кольору, таким чином створюючи кольорову копію окремого символу або слова. Також за допомогою тіні можна зробити текст більш читабельним, якщо контраст між кольором тексту і фону недостатній.

Кожна тінь застосовується як до самого тексту, так і до властивості text-decoration, якщо вона є. Ви можете встановити кілька тіней одночасно, розділяючи їх комами. Тіні перекривають один одного, але не перекривають сам текст. Перша тінь завжди розташовується над рештою тіней.

Властивість text-shadow може мати до чотирьох значень: два або три значення, що визначають довжину тіні, і необов'язковий колір.

text-shadow: x-offset | y-offset | blur | color;
  • x-offset: задає горизонтальний зсув тіні; додатне значення створює зсув тіні праворуч від тексту, а від'ємна довжина створює тінь ліворуч
  • y-offset: задає вертикальний зсув тіні; додатне значення зміщує тінь вниз, від'ємне - вгору
  • blur: задає радіус розмиття; від'ємні значення не допускаються; якщо значення розмиття дорівнює 0, край тіні є чітким; інакше, чим більше значення, тим більше розмивається край тіні

Наприклад:

color: #fff;
text-shadow: 1px 1px #32557f,
             1px -1px #32557f,
             -1px 1px #32557f,
             -1px -1px #32557f,
             3px 3px 6px rgba(0,0,0,.5);

Неоморфізм та Soft UI

Тіні CSS також активно використовуються в неоморфному дизайні.

Неоморфний (також відомий як Neomorphism або Soft UI design) дизайн - це напрямок в дизайні, що означає створення 3D-подібних м'яких форм за допомогою властивості box-shadow.

Неоморфний дизайн характеризується наступними особливостями:

  • Зазвичай це світлі відтінки сірого або відтінки, близькі до білого. Втім, ви можете використовувати будь-який колір, який вам подобається, і це все одно буде неоморфний дизайн.
  • Зазвичай фон має той самий колір (або дуже схожий), що й елементи, які виглядають так, ніби вони пов'язані з фоном.
  • Кути, як правило, закруглені.
  • Тіні. Елементи мають дві тіні - зазвичай темну та білу.
  • Зазвичай рамки відсутні. Однак елементи можуть мати додаткову тонку рамку, щоб зробити краї трохи гострішими та покращити контрастність.

Ось приклад:

По-перше, нам потрібно видалити межі і контур, а також застосувати інші загальні правила CSS до входів і кнопок:

$border_radius: 15px;
$bgr-main: #eee;

input, .btn {
    outline: none;
    border-style: none;
    border-radius: $border_radius;
    background-color: $bgr-main;
    padding: 10px 15px;
    width: 100%;
    height: 40px;
}

Далі нам потрібно додати box-shadow до елементів: тінь буде внутрішньою, щоб зробити елементи порожнистими або втиснутими, або винесеною, щоб зробити елементи опуклими:

.inset {
    box-shadow: inset -4px -4px 10px #fff,
                inset 4px 2px 12px #b9bbc4;
}

.outset {
    box-shadow: 5px 5px 10px #b9bbc4,
                -5px -5px 12px #fff;
}

Ми також можемо трохи змінити тінь кнопки входу, щоб покращити контрастність при наведенні:

.login-btn:hover {
    box-shadow: 5px 5px 5px #b9bbc4,
                -5px -5px 5px #fff;
}
