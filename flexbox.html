Flexbox

Потужний інструмент для розміщення елементів на сторінці веб-сайту.
Динамічні, гнучкі, сучасні та адаптивні макети без використання складних таблиць і фреймів.

Коротко про Flexbox

CSS flexbox (Flexible Box Layout Module) - це модуль верстки, який складається з гнучкого контейнера (батьківського елемента) і гнучких елементів (дочірніх елементів). Флекс-елементи можуть бути організовані у вигляді рядка або стовпця, а вільний простір між ними можна розподілити різними способами.

За допомогою Flexbox ви можете:

  • Автоматично масштабувати елементи (змінювати висоту або ширину) так, щоб вони заповнювали доступний простір
  • Автоматично зменшувати або збільшувати елементи, щоб вони поміщалися в контейнер і не переповнювали його
  • Змінювати порядок елементів
  • Вирішити проблему горизонтального та вертикального центрування
  • Створювати колонки однакової висоти
  • Створювати нижній колонтитул, що прилипає до нижньої частини сторінки
  • Проектувати навігаційні панелі
  • І багато іншого

Створення Flex-контейнера

Властивість display: flex; застосовується до контейнера, робить його блоковим елементом і "вмикає" гнучку верстку для всіх його прямих дочірніх елементів.

.container {
    display: flex;
}

display: inline-flex; працює так само. Тільки він створює контейнер як вбудований (подібний строковому) елемент.

.container {
    display: inline-flex;
}

Cтворення рядка або стовпця

Якщо ви хочете, щоб flex елементи розташовувалися як рядок або стовпець, застосуйте до контейнера властивість flex-direction.

Властивість flex-direction: row; розташує елементи горизонально, у рядок. row - це значення за замовчуванням.

.container {
    flex-direction: row;
}

Якщо у вас є більше елементів, які можуть поміститися в один рядок, але ви все одно хочете горизонтальне розташування, вам стане в нагоді властивість flex-wrap: wrap;. Таким чином, flex елементи перенесуться на декілька рядків всередині flex контейнера. Значення за замовчуванням - flex-wrap: nowrap;.

.container {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Щоб розташувати елементи вертикально, у стовпчик, застосуйте до контейнера flex-direction: column;.

.container {
    flex-direction: column;
}

Існує скорочення для властивостей flex-direction і flex-wrap - flex-flow. Значення за замовчуванням - row nowrap. 

.container {
    flex-flow: row wrap; // flex-direction + flex-wrap
}

Вирівнювання елементів по горизонталі

Щоб задати горизонтальне вирівнювання елементів, використовуйте властивість justify-content. Вона може мати одне з наступних значень:

  • flex-start: елементи вирівнюються по лівому краю контейнера (значення за замовчуванням)
  • flex-end: елементи вирівнюються по правому краю контейнера
  • center: елементи вирівнюються по центру контейнера
  • space-between: елементи рівномірно розподілені в рядку (перший елемент - зліва, останній - справа)
  • space-around: елементи рівномірно розподілені в рядку з однаковим простором по обидва боки від них
  • space-evenly: елементи розподіляються так, щоб відстань між будь-якими двома елементами (і відстань до країв) була однаковою
.container {
    justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
}

Вирівнювання елементів по вертикалі

Щоб вирівняти flex елементи по вертикалі, скористайтеся властивостями align-items, align-content або align-self.

align-items

Щоб визначити вертикальне вирівнювання елементів, застосуйте до контейнера властивість align-items. Ця властивість може мати одне з наступних значень:

  • stretch: елементи розтягуються, щоб заповнити контейнер, дотримуючись властивостей min-width та max-width (це значення за замовчуванням)
  • flex-start: елементи вирівнюються до верхнього краю контейнера
  • flex-end: елементи вирівнюються до нижнього краю контейнера
  • center: елементи вирівнюються по центру контейнера
  • baseline: елементи вирівнюються по їхнім базовим лініям
.container {
    align-items: stretch | flex-start | flex-end | center | baseline;
}

align-content

Щоб визначити вертикальне вирівнювання декількох рядків елементів, використовуйте властивість align-content. Ця властивість не має ефекту, якщо є лише один рядок flex елементів.

Ця властивість працює тільки при виконанні двох умов:

  • Значення flex-wrap повинно бути застосовано до контейнера
  • Flex-контейнер повинен мати більшу висоту, ніж рядки flex-елементів

Іншими словами, всередині контейнера має бути додатковий вертикальний простір, який повинен бути більшим, ніж сума висот усіх рядів елементів.

Ця властивість може мати одне з наступних значень:

  • stretch: рядки розтягуються, щоб зайняти весь доступний простір у контейнері (значення за замовчуванням)
  • flex-start: рядки розміщуються у верхній частині контейнера
  • flex-end: рядки розміщуються у нижній частині контейнера
  • center: рядки відцентровані посередині контейнера
  • space-between: рядки рівномірно розподілені в контейнері
  • space-around: рядки рівномірно розподілені з рівним пробілом навколо кожного рядка
.container {
    align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

align-self

Щоб вирівняти окремі елементи по вертикалі, використовуйте властивість align-self. Вона застосовується до flex елементів, а не до контейнера. Корисно перевизначити вирівнювання, яке вказане для контейнера, за допомогою властивості align-items.

Властивість align-self може мати одне з наступних значень (ті самі, що й у випадку властивості align-items, а також значення auto):

  • auto: дорівнює значенню, вказаному у властивості align-items для контейнера (значення за замовчуванням)
  • stretch: елементи розтягуються, щоб заповнити контейнер, дотримуючись властивостей мінімальної та максимальної ширини
  • flex-start: елементи розміщуються у верхній частині контейнера
  • flex-end: елементи розміщуються в нижній частині контейнера
  • center: елементи розташовуються по центру контейнера
  • baseline: елементи вирівнюються по їхнім базовим лініям
.item {
    align-self: auto | stretch | flex-start | flex-end | center | baseline;
}

Зміна порядку елементів

Кілька властивостей дозволяють визначити порядок елементів у контейнері: order, row-reverse, column-reverse, wrap-reverse.

order

Щоб налаштувати порядок окремих flex елементів, використовуйте властивість order. Вона застосовується до flex елементів.

За замовчуванням, всі flex-елементи мають значення order: 0;. Якщо ви вказуєте значення -1 для елемента, він переміщується на початок рядка, а значення 1 - в кінець. Якщо кілька flex елементів мають однакове значення порядку, вони відображатимуться відповідно до початкового порядку.

.item {
    order: <integer>;
}

Порядок за замовчуванням:

Зробити останній елемент першим:

.item5 {
    order: -1;
}

Змінити порядок окремих елементів:

.item1 { order: 4; }
.item2 { order: 3; }
.item3 { order: 1; }
.item4 { order: 5; }
.item5 { order: 2; }

row-reverse, column-reverse

Ви можете змінити порядок розташування елементів. Щоб розташувати елементи в рядку справа наліво, застосуйте властивість flex-direction: row-reverse; до контейнера.

.container {
    flex-direction: row-reverse;
}

Щоб розташувати flex елементи у стовпчику знизу вгору, використовуйте властивість flex-direction: column-reverse;.

.container {
    flex-direction: column-reverse;
}

Замість flex-direction ви можете вибрати скорочену властивість flex-flow для застосування цих значень:

.container {
    flex-flow: row-reverse wrap;
}

.container {
    flex-flow: column-reverse nowrap;
}

wrap-reverse

Також можна застосувати властивість flex-wrap: wrap-reverse;, яка дозволяє "обгортати" кілька рядків flex елементів знизу вгору.

.container {
    flex-wrap: wrap-reverse;
}

Збільшення або зменшення елементів

Гнучкість flex елементів визначається властивостями flex-basis, flex-grow і flex-shrink.

flex-basis

Властивість flex-basis визначає розмір за замовчуванням для flex елемента перед розподілом доступного простору. Ви можете задати абсолютне значення (наприклад, 200px або 10em), значення у відсотках (наприклад, 50%) або ключове слово (auto, content).

.item {
    flex-basis: <length> | auto | content;
}

Властивість flex-basis можна інтерпретувати як мінімальну ширину flex елемента. Якщо ви вкажете значення flex-basis, воно задасть ширину для конкретного елемента, але залежно від інших параметрів гнучкості, flex елемент може стати ширшим (або вужчим) за значення flex-basis.

Значення за замовчуванням - auto. Воно отримує значення властивості width. А значення content базується на вмісті flex елемента.

Якщо ви не задасте значення ﬂex-basis і встановите значення flex-grow рівним 0, елемент буде стиснуто до мінімального розміру.

flex-grow

Властивість flex-grow застосовується до окремих flex елементів і визначає здатність flex елемента рости. Вона диктує, яку частину доступного простору всередині flex контейнера повинен займати елемент. Вона приймає значення без одиниці, яке слугує пропорцією. Значення за замовчуванням - 0, від'ємні числа не приймаються.

Якщо для всіх елементів встановлено значення flex-grow, рівне 1, решту простору в контейнері буде розподілено порівну між усіма дочірніми елементами. Якщо значення flex-grow для елемента встановлено на 2, він отримає у 2 рази більше пікселів до початкової ширини, ніж інші елементи зі значенням flex-grow рівним 1.

.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
.item3 { flex-grow: 3; }

flex-shrink

Властивість flex-shrink застосовується до flex елементів і вказує, наскільки кожен елемент буде зменшено, якщо не вистачає вільного місця. Вона приймає значення без одиниці, яке слугує пропорцією. Значення за замовчуванням - 1. 0 дозволяє зберегти початковий розмір елемента. Від'ємні значення є неприпустимими.

.container { width: 700px; padding: 20px; }

.item1 { flex-basis: 150px; flex-shrink: 1; }
.item2 { flex-basis: 250px; flex-shrink: 2; }
.item3 { flex-basis: 350px; flex-shrink: 3; }

flex

Властивість flex - це скорочення для параметрів flex-grow, flex-shrink і flex-basis разом узятих. Другий і третій параметри (flex-shrink і flex-basis) є необов'язковими.

Він може мати кілька значень:

  • flex-grow flex-shrink flex-basis: значення трьох властивостей разом узятих
  • none: те саме, що і 0 0 auto (робить flex елементи повністю негнучкими)
  • initial: те саме, що 0 1 auto (значення за замовчуванням; воно базується на властивостях width та height)
  • auto: так само, як 1 1 auto (базується на властивостях width та height, але робить flex елементи повністю гнучкими)
  • <positive number>: так само як <positive number> 1 0 (робить елемент гнучким і встановлює основу flex рівним нулю)

Наприклад, flex: 1 1 300px; ідентичний коду нижче:

flex-grow: 1;
flex-shrink: 1;
flex-basis: 300px;

А flex: 1; ідентичний коду, наведеному нижче:

flex-grow: 1;
flex-shrink: 1;
flex-basis: 0%;

У наведеному вище прикладі ширина flex елемента визначатиметься не його вмістом, а властивістю flex-grow.

body {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

main {
    flex: 1;
}
