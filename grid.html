Grid Layout

Потужний інструмент для створення складних макетів веб-сторінок з гнучкою і адаптивною сіткою.
Створюйте будь-які веб-сторінки, від простих блогів до складних електронних магазинів.

Коротко про Grid Layout. CSS Grid та Flexbox

CSS Grid - це система для створення адаптивних веб-сайтів. За допомогою CSS Grid можна спробувати зробити багато речей, наприклад, галерею зображень, шахову дошку, макет коміксу або меню ресторану.

Як і Flexbox, CSS Grid - це метод створення адаптивного макета. Однак, між ними є різниця: Flexbox призначений для одновимірних макетів, а Grid корисний для створення двовимірних макетів. Якщо ви розміщуєте елементи в одному напрямку (наприклад, кнопки всередині навігаційного елемента), то вам слід використовувати Flexbox. Але якщо ви створюєте цілий макет з рядками і стовпчиками, вам варто вибрати CSS Grid. І, звичайно, ви можете комбінувати ці два варіанти.

У Flexbox ви визначаєте вміст. А за допомогою CSS Grid спочатку потрібно визначити макет (стовпці та рядки). Ви визначаєте ширину стовпців і висоту рядків, і лише після цього розміщуєте вміст у доступних комірках сітки.

Термінологія

Grid Container - це елемент, до якого застосовується display: grid;. Він є прямим батьком усіх елементів сітки.

Grid Item - це дочірні елементи (прямі нащадки), вкладені в батьківський контейнер сітки. Вони можуть бути будь-якими елементами і представляти собою заголовок, бічну панель, нижній колонтитул тощо.

Grid Lines - це розділові лінії, які утворюють структуру сітки. Вони можуть бути вертикальними (лінії сітки стовпців) або горизонтальними (лінії сітки рядків) і розташовуватися з обох боків рядка або стовпця.

Grid Track - це простір між двома сусідніми лініями сітки. Доріжки сітки - це стовпчики або рядки сітки.

Grid Cell - це простір між двома сусідніми рядками і двома сусідніми стовпчиками сітки. Це єдина одиниця сітки. Ви можете розмістити елемент у кількох комірках.

Grid Area - це загальний простір, оточений чотирма лініями сітки. Вона може складатися з будь-якої кількості комірок сітки.

Grid Container

Щоб створити контейнер сітки CSS, застосуйте display: grid; до батьківського елемента <div>. Властивість display може мати ще одне значення - inline-grid, яке генерує сітку на рівні inline.

<div class="container">
    <div class="item1">starfish</div>
    <div class="item2"></div>
    <div class="item3">clown-fish</div>
    <div class="item4"></div>
    <div class="item5">shark</div>
    <div class="item6"></div>
    <div class="item7">jellyfish</div>
    <div class="item8"></div>
    <div class="item9">crab</div>
</div>

.container {
    display: grid;
}

CSS Grid Template

Існує чотири властивості grid-template-*: grid-template-rows, grid-template-columns, grid-template-areas і grid-template. Остання є скороченням для перших трьох.

grid-template-columns та grid-template-rows

Властивості CSS grid-template-columns і grid-template-rows визначають стовпці і рядки сітки.

Їх значення представляють розмір доріжки (може бути довжиною, відсотком або часткою вільного простору в сітці - одиниця fr), а простір між ними є лінією сітки (назва на ваш вибір).

.container {
    grid-template-columns: <track-size> | <line-name> <track-size>;
    grid-template-rows: <track-size> | <line-name> <track-size>;
}

Якщо ваше визначення містить частини, що повторюються, ви можете використати функцію repeat():

.container {
    grid-template-columns: repeat(3, 33.33%);
    grid-template-rows: repeat(3, 33.33% [col-start]);
}

Що еквівалентно цьому:

.container {
    grid-template-columns: 33.33% 33.33% 33.33%;
    grid-template-rows: 33.33% [col-start] 33.33% [col-start] 33.33% [col-start];
}

Це створює три рядки з висотою 33,33% і три стовпчики з шириною 33,33%.

Якщо ви хочете, щоб ширина або висота стовпців чи рядків залежали від розміру вмісту, використовуйте grid-template-columns: auto; або grid-template-rows: auto;.

grid-template-areas

Ця властивість визначає шаблон сітки, посилаючись на назви областей сітки, які вказано за допомогою властивості grid-area. Повторення назви області сітки призводить до того, що вміст поширюється на ці клітинки. Синтаксис візуально представляє структуру сітки.

.container {
    display: grid;
    grid-template-rows: repeat(4, 25%);
    grid-template-columns: repeat(5, 20%);
    grid-template-areas:
        "a b b b b"
        "a c c c d"
        "a c c c d"
        "e f g h i";
}

grid-area дає елементу ім'я, щоб на нього міг посилатися шаблон, створений за допомогою властивості grid-template-areas.

.item2 { grid-area: a; }
.item4 { grid-area: b; }
.item6 { grid-area: c; }
.item8 { grid-area: d; }

.item1 { grid-area: e; }
.item3 { grid-area: f; }
.item5 { grid-area: g; }
.item7 { grid-area: h; }
.item9 { grid-area: i; }

Це створить сітку шириною 5 стовпців і висотою 4 рядки:

"A" - це одна область на 3 клітинки вниз, а "B" - на 4 клітинки праворуч. CSS Grid автоматично розглядає їх як окремі блоки.

Ви можете використовувати крапку як значення площі сітки. Вона позначатиме порожню комірку сітки:

grid-template-areas:
    "a b b b b"
    "a c c c d"
    "a c c c d"
    ". f g h .";

grid-templates

Властивість grid-template - це скорочення для встановлення шаблонів grid-template-rows, grid-template-columns і grid-template-areas в одному оголошенні.

.container {
    display: grid;
    grid-template:
        "a b b b b" 25% // представляє рядок з площами та висотою рядка
        "a c c c d" 25%
        "a c c c d" 25%
        "e f g h i" 25%
        / 20% 20% 20% 20% 20%; // представляє ширину стовпців
}

Це еквівалентно цьому:

.container {
    display: grid;
    grid-template-rows: repeat(4, 25%);
    grid-template-columns: repeat(5, 20%);
    grid-template-areas:
        "a b b b b" 25%
        "a c c c d" 25%
        "a c c c d" 25%
        "e f g h i" 25%;
}

Grid Auto

Існує три властивості grid-auto-*: grid-auto-rows, grid-auto-columns та grid-auto-flow.

У той час як властивості grid-template-* використовуються для визначення положення та розміру клітинок сітки, властивості grid-auto-* використовуються лише для визначення розміру клітинок сітки. На відміну від властивостей grid-template-*, властивості grid-auto-* приймають лише одне значення довжини.

grid-auto-columns та grid-auto-rows

Ці властивості визначають розмір треків сітки, які створюються, коли елементів сітки більше, ніж клітинок у сітці, або коли елемент сітки розміщено поза межами сітки, визначеної властивостями grid-template-* (явна сітка).

Використовуйте властивість grid-auto-columns, щоб встановити ширину за замовчуванням для всіх стовпців. Властивість grid-template-columns перевизначає цю властивість.

Використовуйте властивість grid-auto-rows для встановлення висоти за замовчуванням для всіх рядків. Властивість grid-template-rows перевизначає цю властивість.

grid-auto-columns та grid-auto-rows можуть мати такі значення:

  • auto: встановлює розмір рядків або стовпців на основі розміру контейнера за замовчуванням; комірки з auto розтягуватимуться, щоб заповнити простір, що залишився, відповідно до розміру батьківського контейнера
  • max-content: встановлює розмір кожного рядка або стовпця залежно від найбільшого елемента в рядку або стовпці
  • min-content: встановлює розмір кожного рядка або стовпця залежно від найменшого елемента в рядку або стовпці
  • <length>: задає розмір рядків або стовпців за допомогою значення довжини (px, em, % тощо)
  • minmax(min, max): встановлює діапазон розмірів, більший або рівний min і менший або рівний max (застосовується лише до grid-auto-columns)
  • fr: частка вільного простору в сітці (застосовується лише до grid-auto-columns)
.container {
    display: grid;
    grid-auto-rows: 150px;
}

grid-auto-flow

Якщо у вас є елементи сітки, які не розміщені явно на сітці, ви можете використати властивість grid-auto-flow, яка автоматично розмістить їх на сітці.

Він може мати наступні значення:

  • row: заповнює кожен рядок по черзі, додаючи нові рядки за необхідності (за замовчуванням)
  • column: заповнює кожен стовпчик по черзі, додаючи нові стовпчики за потреби
  • dense: змушує дочірні елементи заповнювати прогалини, якщо вони можуть поміститися (якщо менші елементи з'являться пізніше). Це значення змінює візуальний порядок елементів і може призвести до того, що вони виглядатимуть невпорядковано

Якщо ви встановите grid-auto-flow на column, елементи будуть стікати вниз по стовпчиках:

.container {
    display: grid;
    grid-template-rows: 80px 80px 80px;
    grid-template-columns: 96px 96px 96px 96px;
    grid-auto-flow: column;
}

.item2 {
    grid-column: 1;
    grid-row: 1 / 3;
}

.item6 {
    grid-column: 5;
    grid-row: 1 / 3;
}

Grid Gaps

Проміжки - це горизонтальні та вертикальні проміжки між комірками сітки. Вони визначають розмір ліній сітки. Ви можете створювати проміжки за допомогою властивостей grid-column-gap і grid-row-gap. grid-column-gap: 24px; створює проміжок 24px між стовпчиками, grid-row-gap: 24px; створює проміжок 24px між рядками.

.container {
    display: grid;
    grid-template-rows: repeat(3, 30%);
    grid-template-columns: repeat(3, 30%);
    grid-row-gap: 12px;
    grid-column-gap: 24px;
}

Ви можете використовувати grid-gap як скорочення для grid-row-gap і grid-column-gap.

.container {
    display: grid;
    grid-template-rows: repeat(3, 30%);
    grid-template-columns: repeat(3, 30%);
    grid-gap: 12px 24px;
}

Вирівнювання елементів по горизонталі

Ці властивості вирівнюють сітку вздовж горизонтальної осі: justify-items, justify-content і justify-self.

justify-items

Ця властивість застосовується до всіх елементів сітки всередині контейнера. Властивість justify-items може мати наступні значення:

  • start: вирівнює елементи за початковим краєм комірки
  • end: вирівнює елементи за кінцевим краєм комірки
  • center: вирівнює елементи по центру комірки
  • stretch: заповнює всю ширину комірки (за замовчуванням)
.container {
    justify-items: start | end | center | stretch;
}

justify-content

Іноді загальний розмір сітки може бути меншим за розмір контейнера сітки. Це може статися, якщо всі елементи сітки задано у негнучких одиницях виміру, наприклад, у пікселях. У цьому випадку ви можете вирівняти вміст сітки всередині контейнера сітки за допомогою властивості justify-content.

Він може мати наступні значення:

  • start: вирівнює сітку до початкового краю контейнера сітки
  • end: вирівнює сітку по відношенню до торцевого краю контейнера з сіткою
  • center: вирівнює сітку по центру контейнера сітки
  • stretch: змінює розмір елементів сітки, щоб сітка заповнила всю ширину контейнера сітки (не працює з негнучкими блоками)
  • space-around: розміщує рівну кількість проміжків між кожним елементом сітки, з половинними проміжками на дальніх кінцях
  • space-between: розміщує рівну кількість проміжків між кожним елементом сітки, без проміжків на дальніх кінцях
  • space-evenly: розміщує рівну кількість проміжків між кожним елементом сітки, включно з дальніми кінцями
.container {
    justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
}

justify-self

Щоб вирівняти окремий елемент сітки всередині однієї клітинки, встановіть для нього властивість justify-self. Вона може мати такі значення:

  • start: вирівнює елемент сітки за лівим краєм комірки
  • end: вирівнює елемент сітки за правим краєм комірки
  • center: вирівнює елемент сітки по центру комірки
  • stretch: заповнює всю ширину комірки (за замовчуванням)
.item {
    justify-self: start | end | center | stretch;
}

Вирівнювання елементів по вертикалі

Ці властивості вирівнюють сітку по вертикальній осі: align-items, align-content та align-self.

align-items

Ця властивість застосовується до всіх елементів сітки всередині контейнера. Властивість align-items може мати наступні значення:

  • start: вирівнює елементи за початковим краєм комірки.
  • end: вирівнює елементи за кінцевим краєм комірки
  • center: вирівнює елементи по центру комірки
  • stretch: заповнює всю висоту комірки (за замовчуванням)
.container {
    align-items: start | end | center | stretch;
}

align-content

Іноді загальний розмір сітки може бути меншим за розмір контейнера сітки. Це може статися, якщо всі елементи сітки задано у негнучких одиницях виміру, наприклад, у пікселях. У цьому випадку ви можете вирівняти вміст сітки в контейнері сітки за допомогою властивості align-content.

Він може мати наступні значення:

  • start: вирівнює сітку до початкового краю контейнера сітки
  • end: вирівнює сітку по відношенню до торцевого краю контейнера з сіткою
  • center: вирівнює сітку по центру контейнера сітки
  • stretch: змінює розмір елементів сітки, щоб сітка заповнила всю висоту контейнера сітки (не працює з негнучкими блоками)
  • space-around: розміщує рівну кількість проміжків між кожним елементом сітки, з половинними проміжками на дальніх кінцях
  • space-between: розміщує рівну кількість проміжків між кожним елементом сітки, без проміжків на дальніх кінцях
  • space-evenly: розміщує рівну кількість проміжків між кожним елементом сітки, включно з дальніми кінцями
.container {
    align-content: start | end | center | stretch | space-around | space-between | space-evenly;
}

align-self

Щоб вирівняти окремий елемент сітки всередині однієї клітинки, встановіть для нього властивість align-self. Вона може мати такі значення:

  • start: вирівнює елемент сітки за лівим краєм комірки
  • end: вирівнює елемент сітки за правим краєм комірки
  • center: вирівнює елемент сітки по центру комірки
  • stretch: заповнює всю висоту комірки (за замовчуванням)
.item {
    align-self: start | end | center | stretch;
}

Вирівнювання елементів по горизонталі та вертикалі

Властивості place-* є скороченнями властивостей justify-* та align-*.

place-items

place-items встановлює властивості align-items та в одному оголошенні.

.container {
    place-items: <align-items> <justify-items>;
}

Перше значення задає align-items, а друге - justify-items. Якщо друге значення не вказано, до обох властивостей буде застосовано перше значення.

Приклади:

.container {
    place-items: center;
}

.container {
    place-items: center stretch;
}

place-content

place-content встановлює властивості align-content та justify-content в одному оголошенні.

.container {
    place-content: <align-content> <justify-content>;
}

Перше значення задає align-content, а друге - justify-content. Якщо друге значення не вказано, до обох властивостей буде застосовано перше значення.

Приклади:

.container {
    place-content: center;
}

.container {
    place-content: center stretch;
}

place-self

place-self встановлює властивості align-self та justify-self в одному оголошенні.

.item {
    place-self: <align-content> <justify-content>;
}

Перше значення встановлює align-self, а друге - justify-self. Якщо друге значення не вказано, до обох властивостей буде застосовано перше значення.

Приклади:

.item {
    place-self: center;
}

.item {
    place-self: center stretch;
}

grid

Властивість grid - це скорочена властивість, яка встановлює всі явні властивості сітки (grid-template-rows, grid-template-columns і grid-template-areas) або всі неявні властивості сітки (grid-auto-rows, grid-auto-columns і grid-auto-flow) в одному оголошенні.

Ви можете вказати тільки явні або неявні властивості сітки в одному оголошенні grid.

.container {
    grid: 250px 400px / 4fr 2fr;
}

Еквівалентно:

.container {
    grid-template-rows: 250px 400px;
    grid-template-columns: 4fr 2fr;
}

grid-*-start, grid-*-end

Цей набір властивостей (grid-column-start, grid-column-end, grid-row-start, grid-row-end) визначає розташування елемента сітки в сітці і вказує початкову та кінцеву точки, через які ви хочете протягнути вміст клітинки. Вони посилаються на певні лінії сітки і застосовуються до елемента, а не до контейнера.

  • grid-column-start/grid-row-start: рядок, з якого починається елемент
  • grid-column-end/grid-row-end: рядок, в якому закінчується елемент
  • <line>: номер для посилання на пронумеровану лінію сітки або ім'я для посилання на іменовану лінію сітки
  • span <number>: елемент буде простягатися через вказану кількість доріжок сітки
  • span <name>: елемент буде переміщуватись, доки не досягне наступного рядка з вказаною назвою
  • auto: автоматичний проміжок або проміжок за замовчуванням

Ви також можете використовувати скорочення: grid-column (для grid-column-start + grid-column-end) або grid-row (grid-row-start + grid-row-end). Кожна з них приймає ті ж самі значення, що і повна версія.

Наприклад:

.item {
    grid-row-start: 2;
    grid-row-end: 4;
}

Або:

.item {
    grid-row: 2 / span 3;
}

Це означає, що елемент займе рядки з 2 по 4. Якщо це записано як grid-row: 2 / span 3;, перша частина - це рядок, з якого починається елемент, а друга частина - це кількість клітинок, через які він пройде. Це працює так само для стовпців.

Спеціальні ключові слова та функції

Для визначення розмірів рядків і стовпців можна використовувати різні довжини: px, rem, % тощо. Ви також можете використовувати такі ключові слова, як auto-fit, auto-fill, mi-content, max-content, дробові одиниці (fr), а також такі функції, як minmax() і repeat().

auto-fill заповнює рядок стількома стовпчиками, скільки він може вмістити. Він створює стовпці щоразу, коли новий стовпець може поміститися. Новостворені стовпці можуть бути порожніми, але вони все одно займатимуть відведене місце в рядку.

.container {
    grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
}

auto-fit вписує доступні на даний момент стовпці в простір, розширюючи їх так, щоб вони займали будь-який вільний простір. Це дозволяє обгортати стовпці в рядки, коли не вистачає місця для мінімального значення, не переповнюючи вміст. Нові додані стовпці не займатимуть місця і будуть згорнуті.

.container {
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
}

Дробова одиниця (fr) - це одна частина доступного простору. 1fr вказує браузеру розподілити простір між колонками так, щоб кожна колонка отримала рівну частину цього простору. Якщо ширина div дорівнює 400px, repeat(4, 1fr) означає, що 400px буде розділено на 4 рівні частини по 100px. Ви також можете використовувати числа з плаваючою комою для визначення fr. Той самий 1fr в межах одного батька зменшиться до меншого розміру, коли будуть додані пропуски.

.container {
    grid-template-columns: repeat(4, 1fr);
}

min-content представляє найбільший внесок min-content елементів сітки, що займають доріжку сітки. Внесок min-content - це розмір, який вносить блок до розміру min-content блоку, що його містить.

.container {
    grid-template-columns: 200px 1fr 2fr min-content;
}

max-content представляє найбільший внесок максимального вмісту елементів сітки, що займають доріжку сітки. Внесок максимального вмісту - це розмір, який вносить блок до розміру максимального вмісту блоку, що його містить.

.container {
    grid-template-columns: 200px 1fr 2fr max-content;
}

minmax(min, max) встановлює діапазон розмірів, більший або рівний min і менший або рівний max. Наприклад, щоб встановити розмір стовпця 1fr, але зменшити його не більше ніж до 300px:

.container {
    grid-template-columns: 1fr minmax(300px, 1fr);
}

repeat() дозволяє повторити стовпці або рядки стільки разів, скільки потрібно. Наприклад, щоб зробити 5 стовпців:

.container {
    grid-template-columns: repeat(5, 1fr);
}

Що еквівалентно цьому:

.container {
    grid-template-columns: 1fr 1fr 1fr 1fr 1fr;
}

Ви можете поєднувати ці речі, наприклад:

.container {
    grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
}
